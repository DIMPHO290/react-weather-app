Weather App is a web application built with React and TypeScript. It allows users to check real-time weather updates for any location worldwide. The app provides detailed information such as current conditions, temperature, humidity, wind speed, and multi-day forecasts, helping users stay prepared for the day and week ahead.

Technologies Used

React
TypeScript
CSS
Vite

Setup Instructions

git clone https://github.com/DIMPHO290/react-weather-app.git
cd react-links-vault
npm create@vite.4.1.0 .
npm run dev
